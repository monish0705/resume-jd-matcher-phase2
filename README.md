\# Resume \& Job Description Matcher (Phase 2)



This project is a beginner-friendly AI tool that compares a resume PDF

with a job description and explains how well they match.



\## Tech Stack

\- Python

\- Streamlit

\- pdfplumber

\- OpenAI API



\## Learning Goal

Understand how a single LLM prompt can compare two pieces of text

semantically without using RAG or agents.
